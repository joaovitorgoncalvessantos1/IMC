# 🧮 Calculadora de IMC

Aplicação web simples que calcula o **Índice de Massa Corporal (IMC)** a partir da altura e peso do usuário.
O projeto foi desenvolvido para praticar **JavaScript, manipulação de DOM e lógica de programação**.

---

## 🚀 Funcionalidades

* Inserir **peso e altura**
* Calcular automaticamente o **IMC**
* Exibir a **classificação do IMC**
* Interface simples e responsiva

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
